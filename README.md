## 🛠️ Tech Stack

This project is built using the following technologies:

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Language:** TypeScript / JavaScript
* **Styling:** Tailwind CSS / Shadcn UI / Tailwind CSS
* **Backend Support:** Django REST Framework (DRF)
* **Deployment:** Netlify

---

## ✨ Features

* **Responsive Design:** Optimized for all screen sizes (Mobile, Tablet, Desktop).
* **Modern UI/UX:** Clean, dark/light mode friendly, and highly performant dashboard elements.
* **Dynamic Sections:** Showcasing recent projects, freelance services, and technical expertise.
* **SEO Optimized:** Built with structural metadata and fast Loading speed.
