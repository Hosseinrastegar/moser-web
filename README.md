## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | Next.js 15 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| Animations | CSS Keyframes + Intersection Observer API |
| Icons | Lucide React |
| Font | Vazirmatn (Google Fonts) |

---

## ✨ Features

- **High Performance** — Lighthouse score 90+ on both mobile and desktop
- **Fully Responsive** — Optimized for mobile, tablet, and desktop
- **Smart Animations** — Scroll-triggered reveal on all devices; heavy effects limited to desktop only
- **Glassmorphism UI** — Modern design with backdrop blur and transparency effects
- **SEO Optimized** — Structured metadata and fast load speed
- **Full RTL Support** — Complete Persian language and right-to-left layout support
- **Custom 404 Page** — Animated not-found page with quick navigation
- **Contact Form** — Direct communication via Formspree integration
